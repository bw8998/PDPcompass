# Roadmap

## Milestone 1: Model Architecture, Sample Data Collection & UI Prototype

Duration: Month 1

Goals:

- define provider personas
- define cost model
- define revenue scenario model
- define readiness scoring framework
- collect initial sample inputs
- create UI prototype
- publish methodology draft

Outputs:

- provider persona framework
- cost / revenue model
- PDP readiness framework
- sample data collection structure
- UI prototype
- GitHub methodology draft

## Milestone 2: Simulator MVP, External Testing & Public Launch

Duration: Month 2

Goals:

- build public web simulator
- implement provider profile selection
- implement cost and workload inputs
- implement break-even calculation
- implement readiness score
- run external testing
- launch public simulator

Outputs:

- live PDP Compass simulator
- 20+ external test submissions
- methodology notes
- public launch
