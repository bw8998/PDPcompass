# PDP Provider Cost & Readiness Survey Draft

This draft survey is used to collect anonymized sample inputs from machine owners, miners, infrastructure operators, and PDP-interested participants.

## Operator Profile

- Existing Filecoin SP
- Former miner
- Web3 infrastructure operator
- Small IDC / regional cloud provider
- NAS / homelab operator
- AI server / storage operator
- Other

## Hardware

- CPU model / core count
- RAM
- SSD / NVMe capacity
- HDD capacity
- Usable storage capacity
- Existing hardware or new hardware
- Machine location

## Network

- Upload bandwidth
- Download bandwidth
- Public IP availability
- Domain / HTTPS endpoint availability
- Expected uptime
- Network type: home / office / IDC / cloud

## Cost

- Electricity cost
- Bandwidth cost
- Hosting / rack cost
- Hardware purchase cost
- Maintenance cost
- Estimated monthly operator time
- Depreciation period

## PDP Interest

- Are you interested in running PDP?
- What capacity would you start with?
- What is your minimum expected monthly revenue?
- What is your biggest concern?

Concern options:

- no clear revenue
- setup complexity
- hardware cost
- bandwidth cost
- no workload
- payment uncertainty
- tooling instability
- uptime requirement

## Feedback

- Would you test PDP Compass?
- Can your feedback be anonymized?
- Can we contact you for follow-up?
